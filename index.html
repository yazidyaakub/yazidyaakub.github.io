Featured Projects

Production-grade tools and experiments in geospatial data engineering.

malaysia-geo-catalog

A static geospatial data catalog for exploring Sentinel-2 L2A scenes over Malaysia. It uses Microsoft Planetary Computer STAC data, Python preprocessing, GeoParquet artifacts, MapLibre rendering, state-level filtering, cloud/date filters, MGRS tile extraction, and a dark operational UI.

STAC Sentinel-2 MapLibre GeoPandas GeoParquet Python Static Web
🤖
AI Earth Observation

ai-agent-for-eo-data

An AI-powered Earth observation assistant for querying and analyzing satellite imagery through natural language. It integrates the OpenAI Agents SDK with Planet APIs to search, download, visualize, and analyze imagery, including RGB rendering and vegetation indices such as NDVI and NDRE.

Python OpenAI Agents SDK Planet SDK Rasterio Streamlit GeoJSON Remote Sensing
📊
Spatial Analysis

malaysia-air-pollution-wealth

A geospatial analysis of PM2.5 pollution exposure and wealth patterns across Malaysia using satellite-derived pollution data and Meta Relative Wealth Index data. The project uses spatial matching and interactive visualizations to explore environmental inequality.

Python Pandas Xarray SciPy Plotly Leaflet Spatial Analysis
